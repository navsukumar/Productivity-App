Project description

This is a Todo List application that can add and clear todos. If bored, there is an option to click a "bored button" which will generate a fun task for you to do!

How I made it

This project was made using HTML, CSS, and Java Script. HTML was used to build the DOM, CSS was used to style the DOM elements, and Java Script was used to handle the API interactions and implement the functionalities of each of the buttons.
